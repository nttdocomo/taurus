name

Properties:
Name Type Description
name string

域的name。当form submit()时包含该域的值时它将被用作参数名。如果没有设置name，该值会回滚变成Base#inputId。如果不希望表单提交时包含域，将Base#submitValue设置为false。

valueToRaw(value)

将混合型的值转化为适合在域里展示的原始的表现。它允许控制传给#setValue的值如何展示给用户，包括本地化。例如，对于一个Ext.form.field.Date，它将控制传给#setValue日期对象如何转换为一个String并显示在域里。

关于反向转换查看#rawToValue。

基本的做法是简单进行一次标准的toString转换，并将empty values转化为一个空字符串。
