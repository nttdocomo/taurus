Class: Label

Label

new Label()

生成一个可以插入表单并和表单里的一个字段通过#forId关联起来的单独的<label />元素。

Members

autoFitErrors

Properties:
Name Type Description
autoFitErrors Boolean

Whether to adjust the component's body width to make room for 'side' error messages.

fieldLabel :String

字段的标记。和Label#labelSeparator，以及其位置和尺寸一起被添加。

Type:
  • String

labelAlign :String

控制Label#fieldLabel的位置和对齐。可用的值有：

  • left (默认值) - 标签被放置在字段的左边，并且文字左对齐。它的宽由Label#labelWidth配置决定。
  • top - 标签被放置在字段的上面。
  • right - 标签被放置在字段的右边，并且文字右对齐。它的宽由Label#labelWidth配置决定。
Type:
  • String
Default Value:
  • left

labelSeparator

Properties:
Name Type Description
labelSeparator String

插入到标签文字结束处的符号。设置为空字符串将分隔符完全隐藏。

labelWidth :Number

以像素为单位的Label#fieldLabel的宽。只有在Label#labelAlign被设置为"left"或者"right"时。

Type:
  • Number

msgTarget

错误信息文字显示的位置。必须是一下其中之一的值：

Properties:
Name Type Description
msgTarget string
  • title 在浏览器默认的titie属性弹窗里显示信息。
  • under 在字段的下方添加一个包含错误信息的块级div。
  • side 在字段的右侧添加一个错误图标，在悬停的弹窗里显示信息。
